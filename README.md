# Prediccion_cupo_credito
En el siguiente proyecto se diseñará un modelo de machine learning con la capacidad de predecir el cupo de crédito de un cliente de tarjetas. Basado en comportamientos de compra anteriores.  Se probarán diferentes modelos y es escogerá el que nos presente un mejor rendimiento. Como métrica de medición usaremos la raíz del error cuadrático medio

El proyecto se estará desarrollando usando python como lenguaje de programación y algunas librerías como pandas, NumPy, Matplotlib, Seaborn y Scikitlearn. Los datos fueron tomados del siguiente enlace https://www.kaggle.com/datasets/mahnazarjmand/customer-segmentation
